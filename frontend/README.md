# Film Service Application

Цей проєкт складається з Backend та Frontend частин для управління колекцією фільмів та режисерів.

## Структура проєкту

* **backend/** - Серверна частина на Java (Spring Boot)
* **frontend/** - Клієнтська частина на React

## Як запустити

### Backend
1. Перейдіть у папку `backend`.
2. Налаштуйте базу даних у `application.properties`.
3. Запустіть проєкт через IntelliJ IDEA або команду `mvn spring-boot:run`.
4. Сервер працюватиме на порту `8080`.

### Frontend
1. Перейдіть у папку `frontend`.
2. Встановіть залежності: `npm install`.
3. Запустіть проєкт: `npm start`.
4. Відкрийте браузер за адресою `http://localhost:3050`.
